有序列表使用数字并加上 . 号来表示，如：

1
2
3

2. 第二项
3. 第三项

  1. 第一项
  2. 第二项
  3. 第三项

你也可以用 ``` 包裹一段代码，并指定一种语言（也可以不指定）
